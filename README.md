# QtNicoLiveLib

QtNicoLiveLib is a cross-platform "Nico Nama" library.

QtNicoLiveLib はクロスプラットフォームなニコニコ生放送用ライブラリです。
[Viqo](https://github.com/diginatu/viqo) のコアライブラリです。
（現在移行中）

* クロスプラットフォーム
* Qt5 / C++11
* オープンソース (MIT license)

QtはLGPLのライセンス下です。詳しくは [Qt Home](http://www.qt.io/)へ。
